主要功能

  • 玩家画像测试 - 回答几个简单问题，了解你的训练家类型
  • 个性化首页 - 根据你的偏好定制的首页体验
  • 智能推荐 - 基于你的画像推荐宝可梦
  • 宝可梦详情 - 查看种族值、能力雷达图等详细信息
  • 多维度浏览 - 按世代、属性、分类浏览所有宝可梦
  • 收藏管理 - 收藏你喜欢的宝可梦，创建专属收藏夹
  • 搜索筛选 - 快速找到你想找的宝可梦

说明

本网站使用宝可梦相关的CSV数据文件来展示宝可梦信息。您需要将宝可梦数据文件放置在 assets/js/data/pokedex.csv 位置才能完整使用所有功能。

宝可梦图片资源需要放置在 assets/images/pokemon/ 目录下。

技术栈

  • HTML5 + CSS3 + JavaScript (ES6+)
  • Chart.js (数据可视化)
  • LocalStorage (本地数据存储)
